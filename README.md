# Namaz Bar

A lightweight macOS menu bar application that displays the next prayer time and countdown using the Hujjat.org prayer timetable API.

Designed for Shia Jafari prayer calculations with quick access directly from the macOS menu bar.

---

## Features

• Next prayer countdown directly in the menu bar  
• Full daily timetable in the dropdown  
• Multiple cities supported  
• Notifications before prayer time  
• Offline fallback if the API is unavailable  
• Lightweight and runs silently in the background  

Supported cities:

- London
- Birmingham
- Leicester
- Peterborough

## Installation

1. Download the ZIP from the Releases page
2. Extract the file
3. Drag **Namaz Bar.app** into your **Applications** folder
4. Launch the app

Because the app is not yet code signed, macOS may block it on first launch.

If that happens:

1. Right-click the app
2. Select **Open**
3. Click **Open** again

After this the app will run normally.

---

## Usage

Once launched, the mosque icon will appear in your macOS menu bar.

The menu shows:

- Next prayer
- Countdown timer
- Full daily timetable
- City selection
- Refresh option

The timetable automatically refreshes daily.
